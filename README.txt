This folder contains the simulation code used in the paper "Mating preferences of selfish sex chromosomes" [Muralidhar 2019, DOI:10.1038/s41586-019-1271-7]. 
Each file contains code for the simulation of autosomal, W-linked, X-linked, or Z-linked mate preferences. 
For additional details on the model, please see the Methods section in the paper.
